# Hisham Haris — Premium Cinematic Portfolio

A high-end, cinematic portfolio website built with React, TypeScript, and Tailwind CSS. Featuring a dark-themed aesthetic with vibrant glassmorphism, smooth micro-animations, and full-screen project showcases.

**Live Site:** [https://hisham-haris-portfolio.netlify.app/](https://hisham-haris-portfolio.netlify.app/)

## 🚀 Features

- **Cinematic Experience**: Immersive dark-mode design with smooth GSAP-powered scroll choreographies.
- **Project Showcase**: High-fidelity cards with full-page screenshot previews on hover.
- **Integrated Blog**: Fully managed CMS using Decap CMS (formerly Netlify CMS) for easy content updates.
- **Dynamic Contact Form**: Built-in Netlify Forms for reliable lead capture.
- **Responsive Design**: Optimised for all screen sizes, from mobile to 4K displays.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) & [Framer Motion](https://www.framer.com/motion/)
- **CMS**: [Decap CMS](https://decapcms.org/)
- **Hosting**: [Netlify](https://www.netlify.com/)

## 📂 Project Structure

- `/src`: Core application logic and components.
- `/public`: Static assets, including images and CMS configuration.
- `/public/admin`: Decap CMS dashboard and configuration.
- `/scripts`: Custom build scripts for indexing blog content.

## 🛠️ Development

To run the project locally:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## 📝 Blog Management

The blog is powered by Markdown files and a custom indexing script. To manage posts:
1. Navigate to `/admin` on the live site.
2. Log in via Netlify Identity.
3. Use the visual editor to create or edit posts.
